# Sky NBS Player

## 编译方式
&emsp;本项目使用MinGW32-x64编译，编译目标为x64架构的Windows 10桌面应用。
将仓库clone至本地，切换至仓库根目录并新建名为`dist`的文件夹，运行mingw32-make.exe即可。
